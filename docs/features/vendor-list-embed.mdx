---
title: "Embed Vendors List"
description: "Embed up-to-date vendor information on your website."
---

Copy and paste the following script into the `<head>` section of your website

```html
<script src="https://cookiechimp.com/portal/{account_id}/vendors.js"></script>
```

Add the following div where you want to embed the vendor list.

```html
<div id="cookiechimp-vendor-list"></div>
```
