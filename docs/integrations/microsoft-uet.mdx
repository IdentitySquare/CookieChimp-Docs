Microsoft UET (Universal Event Tracking) Consent Mode integration allows CookieChimp to communicate user consent preferences to Microsoft Advertising's tracking system. This ensures that Microsoft UET tags respect user privacy choices while maintaining advertising campaign effectiveness for consenting users.

## What is Microsoft UET?

Microsoft Universal Event Tracking (UET) is Microsoft Advertising's tracking solution that enables:

- **Conversion tracking** for Microsoft Ads campaigns
- **Audience creation** based on website behavior
- **Campaign optimization** using conversion data
- **Attribution analysis** across Microsoft's advertising network

## How UET Consent Mode Works

When integrated with CookieChimp, Microsoft UET Consent Mode:

1. **Receives consent updates** from CookieChimp in real-time
2. **Adjusts tracking behavior** based on user preferences
3. **Maintains compliance** with privacy regulations
4. **Preserves campaign data** for consenting users

## Prerequisites

Before enabling Microsoft UET Consent Mode:

- **Microsoft Advertising account** with UET tag configured
- **UET tracking code** properly installed on your website
- **CookieChimp advertising category** configured for user consent
- **Conversion goals** set up in Microsoft Advertising

## Consent Categories and UET Features

| Consent Category | UET Feature | Consented Behavior | Declined Behavior |
|-----------------|-------------|-------------------|-------------------|
| **Advertising** | Conversion Tracking | Full conversion attribution | No conversion tracking |
| **Advertising** | Remarketing | Audience building enabled | No audience data collection |
| **Analytics** | Enhanced Conversions | Additional conversion data | Basic conversion only |
| **Personalization** | Custom Audiences | Personalized audience creation | No personalized targeting |

## Verification and Testing

### Using Microsoft Advertising Dashboard
1. **Open Microsoft Advertising** campaign dashboard
2. **Check Conversion Tracking** for recent activity
3. **Verify UET Tag Status** shows healthy tracking
4. **Monitor audience sizes** to confirm consent-based data collection

### Using Browser Developer Tools
1. **Open Network tab** in browser developer tools
2. **Look for bat.bing.com requests** when consent is granted
3. **Verify no UET requests** when advertising consent is denied
4. **Check request parameters** for consent status

### Testing Consent Scenarios

#### Full Advertising Consent
- UET tag fires normally
- Conversion tracking works
- Audience building is active
- All Microsoft Ads features enabled

#### Advertising Consent Denied
- UET tag is blocked or limited
- No conversion attribution
- No audience data collection
- Privacy preferences respected

<Info>
  Microsoft UET Consent Mode automatically adjusts tracking behavior based on user consent. No additional configuration is required once the integration is enabled.
